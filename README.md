## Laboratory Utilization App

## ✨ Features

- Subject Scheduler
- Subject Laboratory Utilizations
- Weekly Utilizations and Attendance Reports
- Term Utilization Reports
- Laboratory Borrowes Card
- Laboratory Reservation (On-going)
- E-Stock Card
- Data Analytics (On-going)

## 🚀 Technology

- React-Vite
- React Router v6
- React Dropzone
- React Icons
- Zustand
- TanStack Query v4
- TailwindCSS
- ShadcnUI
- PostgreSQL
- Node Mailer
- Syncfusion v28 - React Scheduler

## 🧞 Commands

All commands are run from the root of the project, from a terminal:

| Command         | Action                                       |
| :-------------- | :------------------------------------------- |
| `npm install`   | Installs dependencies on both client and api |
| `npm run dev`   | Starts local dev server at `localhost:5173`  |
| `npm run build` | Build your production site to `./dist/`      |
